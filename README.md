# sort-js

sorta cool

## usage

get the code:

    $ git clone https://github.com/jaredmcdonald/sort-js.git
    
use it:

    var sort = require('sort-js')
    var sorted = sort.mergeSort([6,5,4,3,2,1]) // [1, 2, 3, 4, 5, 6]

## tests

install dependencies (mocha and subdependencies):

    $ npm install

then: 

    $ npm test
